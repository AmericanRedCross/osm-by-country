OSM Research Visualization

The State of OpenStreetMap Data

Created by interns for the American Red Cross' GIS team, this site provides the user with some idea of the 'completeness' of the OSM map of the world.

The data has been visualized by utilizing multiple chloropleth maps, each with a five class color scheme that is automatically quantized using d3.js.

To switch maps, simply select one of the options below. To view more detailed information about each country, simple hover your cursor over the country of interest to you.

  • Number of Buildings Mapped
  • Population per Building
  • Roads Mapped (km)
  • Number of HOT Tasks
  • Geofabrik Extract Size
  • Population per MB of Geofabrik Extract
  • Area per MB of Geofabrik Exract
  • Population Density
  • Area
